Detailed Description

template<typename R, typename P, typename FctImplem>
class pixled::api::UnaryFunction< R, P, FctImplem >

Function implementation that take a single argument.

The argument is provided to the constructor as a generic api::Function<P> implementation.

UnaryFunction constructor.

arg must be an api::Function that represents the parameter of this UnaryFunction. Since arg is itself a function, this mechanism can be used to recursively build complex scripts, such as LED animations.

While Fct is likely to be automatically deduced, arg might be one of the following:

  • a constant of type T. In this case, f is built as a Constant.
  • Any object extending the api::Function<P> interface. For example, an other UnaryFunction, a BinaryFunction, a TernaryFunction, or anything else that implements the api::Function<P> interface.
